# BUG: Missing Lidar Modes

**Date**: 2026-03-03

**Description**:
The `truePhysicalLidarHorizontal` and `truePhysicalLidarVertical` styles/modes have disappeared when cycling through modes in the browser client lidar view.

**Expected Behavior**:
Users should be able to cycle through all available Lidar modes, including the true physical modes.

**Current Behavior**:
The modes are skipped or no longer available in the cycle rotation.
