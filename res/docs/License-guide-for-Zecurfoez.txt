License Map for Zecurfoez contributions

This document attempts to specify:
- Which license should Zecurfoez contribution have?
- How does project provide attribution & credit to contributors?
- Licensed content

SOURCE CODE LICENSES
For source code the project license is GPLv3, so thats for all game logic
and scripts: https://www.gnu.org/licenses/gpl-3.0.en.html

In effect, the contributions in this repository are already covered by
GPLv3 as stated in ./LICENSE -file.


ARTWORK, GRAPHIC DESIGNS, MUSIC, AUDIO, WRITTEN WORKS
Currently project contributors should use GPLv3 or Creative Commons 4.0
Attribution-ShareAlike 4.0 International -licenses.

Where contributors so desire, or where GPLv3 otherwise invalid, contributions
may be licensed with CC-BY-NC-SA-4.0 license. This allows free reuse in 
opensource projects - but no commercial use:
http://creativecommons.org/licenses/by-nc-sa/4.0/

CC-BY-NC-SA-4.0 licensed contributions must be in separate files, with license
statemant at top of the file: "Creative Commons Attribution-NonCommercial-ShareAlike
4.0 International License." 


CONTRIBUTOR ATTRIBUTION AND CREDIT
Currently the Zecurfoez repository branch Master at
Github.com/csmr/zecurfoez is the source-of-truth for contributors identity.

Contributors maintain their copyright and authorship rights indefinately, and
Zecurfoez project members will in no way claim ownership or copyright, only
reuse license within GPLv3 licensing rules.

All contributors will be mentioned in title credits. Contributor list is
sourced from the repository committers and will (can) be compiled with
./src/make-contributor-list.sh, which will create ./res/contrib-list.txt.
